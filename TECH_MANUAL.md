# Technician User Manual

## Getting Started
1. **Sign Up**: Use the tech signup form with your company email
2. **Login**: Enter your credentials on the login page
3. **Dashboard**: Access the Tech Panel from the main interface

## Core Features

### 1. Barcode Scanner
- **Access**: Click "Barcode Scanner" tab in Tech Panel
- **Camera Scan**: Click "Start Camera" to scan barcodes
- **Manual Entry**: Type barcode numbers directly
- **Search Items**: Scanner searches existing inventory automatically
- **Add New**: If item not found, you can add it to inventory

### 2. Inventory Search
- **Search Bar**: Enter item name, barcode, or description
- **Filter Results**: Results update as you type
- **View Details**: Click items to see full information
- **Stock Levels**: Green (good stock), Yellow (low stock), Red (out of stock)

### 3. Request Items
- **Create Request**: Click "Request Items" tab
- **Select Items**: Choose from inventory list
- **Set Quantity**: Specify how many you need
- **Add Notes**: Include special instructions
- **Submit**: Request goes to admin for approval

### 4. Receive Items
- **Scan Items**: Use barcode scanner to receive shipments
- **Update Quantities**: Confirm received amounts
- **Mark Complete**: Finalize the receiving process

### 5. Mobile Features
- **Responsive Design**: Works on phones and tablets
- **Touch-Friendly**: Large buttons for easy mobile use
- **Offline Ready**: Basic functionality works without internet

## Tips for Efficiency
- Use barcode scanner for fastest item lookup
- Check stock levels before requesting items
- Add detailed notes to requests for faster processing
- Keep the app open during field work for quick access
